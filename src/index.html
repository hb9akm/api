Normalized API for HAMs

The main site of this project is at hamdb.io. This is the more technical page, only about the API part.

The goal of this project is to provide a simple RESTful API to get normalized data of relevance for HAMs.

The main problem this solves is the collection and normalization of data.

This is open source, free to use and comes with no warranty.

Participation is highly encouraged. Please open an issue on GitHub to do so!

Version 2 (beta)

Version 2 aims to unify access accross different endpoints and fix a few shortcomings of v1. Documentation for version 2 can be found here.
